int changeDirectory ( char *  args[])

Cambia el directorio de trabajo actual.

Esta función cambia el directorio de trabajo actual de la shell al especificado en args[1]. Si no se proporciona un directorio, cambia al directorio inicial.

Parameters
argsArgumentos de la línea de comandos. args[1] debe contener el directorio de destino.
Returns
int 0 si el cambio fue exitoso, -1 si hubo un error.
